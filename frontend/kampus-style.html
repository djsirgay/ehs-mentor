Единые стандарты, компоненты и принципы для разработки интерфейсов. Система помогает команде работать быстрее и создавать согласованный пользовательский опыт.

Принципы проектирования

Основные идеи, которые определяют наш подход к дизайну интерфейсов.

Каждый элемент интерфейса должен иметь четкую цель. Убираем лишнее, оставляем только то, что помогает пользователю решить задачу. Простота — это не примитивность, а продуманность каждой детали.
Одинаковые элементы работают одинаково во всех частях продукта. Пользователь изучает интерфейс один раз и применяет знания везде. Это снижает когнитивную нагрузку и повышает эффективность.
Интерфейс должен быть понятен и удобен для всех пользователей, независимо от их возможностей и устройств. Учитываем контрастность, размеры элементов, навигацию с клавиатуры и работу со скринридерами.

Типографика

Система шрифтов и текстовых стилей для создания иерархии и улучшения читаемости.

Используем системные шрифты для быстрой загрузки и естественного восприятия на разных платформах. Основа — system-ui с фоллбэками на -apple-system и sans-serif. Размеры масштабируются адаптивно через clamp().
H1 для главного заголовка страницы, H2 для разделов, H3 для подразделов. Размеры и отступы настроены для создания четкой визуальной иерархии. Межстрочное расстояние уменьшается с увеличением размера шрифта.
